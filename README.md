# Airplane Racing Multiplayer

Мультиплеерная игра-гонка на самолетиках с чекпоинтами, созданная на Unity.

## 🚀 Фичи

1. Создание лобби с настройками: количество игроков, приватное/публичное, имя комнаты.  
2. Функционал лобби:  
   - Отображение подключенных игроков  
   - Возможность покинуть лобби  
   - Отображение кода для подключения  
   - Кнопка начала игры активируется у хоста, когда все игроки подключены  
3. Браузер лобби:  
   - Список открытых лобби  
   - Подключение к приватному лобби по коду  
4. Геймплей:  
   - После старта игры хостом загружается сцена гонки  
   - Начинается отсчет, когда все игроки загрузились  
   - Игроки проходят 10 чекпоинтов  
   - Счёт каждого игрока синхронизируется на всех клиентах  
   - После прохождения всех чекпоинтов отображается победитель  
   - Через 5 секунд все игроки возвращаются в экран лобби  

## 🛠️ Технологии

- Unity  
- Netcode for GameObjects  
- VContainer (Dependency Injection)  
- Unity Multiplayer Services: Session, Relay, Authentication
